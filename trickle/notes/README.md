# Casa Mexa - Sitio Web de Restaurante

## Descripción
Sitio web para restaurante de comida típica mexicana con diseño moderno y funcional.

## Características
- Navbar sticky transparente que cambia a blanco al hacer scroll
- Carrito de compras con contador de productos
- Hero section con imagen de fondo
- Navegación: Inicio, Lo más pedido, Menú, Paquetes, Promociones, Contacto
- Sección "Nuestra Historia" con logros y reconocimientos
- Sección "Lo más pedido" con platillos populares
- Menú con botones colapsables por categorías (Comida, Bebidas, Postres)
- Funcionalidad "Agregar al carrito" con notificaciones
- Sección de Paquetes con combos especiales
- Tarjetas animadas con efecto glass
- Ofertas especiales destacadas
- Sección de Reseñas con estrellas de puntuación
- Mapa de ubicación en sección de contacto
- Botón flotante de WhatsApp para atención al cliente
- Diseño responsivo
- Paleta de colores: Rojo (#C41E3A) y Naranja (#FF8C00)

## Estructura
- `index.html` - Página principal
- `app.js` - Componente principal de la aplicación
- `components/` - Componentes reutilizables
  - Navbar.js - Navegación sticky
  - Hero.js - Sección hero con imagen de fondo
  - MenuCard.js - Tarjetas de platillos
  - PackageCard.js - Tarjetas de paquetes
  - PopularSection.js - Sección de platillos más pedidos
  - PackagesSection.js - Sección de paquetes
  - SpecialOffers.js - Ofertas especiales
  - MenuSection.js - Secciones de menú
  - WhatsAppButton.js - Botón flotante de WhatsApp
  - Footer.js - Pie de página

## Tecnologías
- React 18
- TailwindCSS
- Lucide Icons
